Not another todo app

FlowState doesn't just list your tasks. It classifies them, routes them to the right system, and executes them autonomously.

01

Async Execution

Tasks run in the background. No dialogs, no interruptions. Add a task, keep working — it completes silently.

02

Auto Classification

Every task is instantly categorized — work, system, entertainment, communication — and routed to the right MCP server.

03

Mac Automation

Calendar, Mail, Finder, Apple Music, System Settings, Reminders, Notes — 200+ macOS actions via AppleScript.

04

Browser Control

Navigate websites, fill forms, post tweets, interact with any web app — using your existing logged-in sessions.

05

Focus Tracking

Built-in Flow Mode and Pomodoro timer. All focus time and completed tasks sync to your daily stats automatically.

06

Notch-Native UI

Lives in the menu bar. Drops down from the notch like it belongs there. Dismisses when you click away.

Three keystrokes

Option+Space opens FlowState. Type what you need. Press Return. Done.

1

Type a task in natural language

"Schedule a meeting with John tomorrow at 2pm" or "Play Bohemian Rhapsody" or "Turn on Dark Mode" — just describe what you want done.

2

Claude classifies and routes

The task is analyzed instantly. Calendar events go to Mac Automator. Web tasks go to Playwright. Shortcuts go to Apple Shortcuts. The right tool, every time.

3

Executed silently in the background

No windows pop up. No focus stolen. The task completes while you keep working. Check the result when you're ready.
